# Todo App 

En simpel todo-app byggd med react med lite extra funktioner med möjlighet att lägga till en todo, redigera, radera och checka av och utöver det har vi även med väder för Malmö med aktuell datum och tid.


## Klona projektet

git clone [https://github.com/lalejandroorozcov-sketch/grupp-projekt.git]
cd grupp-projekt

## Installation

### Backend
cd Backend
npm install

### Frontend
cd ../frontend
npm install

## Konfiguration

### Firebase Setup
Placera `serviceAccountKey.json` i backend-mappen

### Portar
- Backend server: port 8080
- Frontend dev server: port 5173

## Starta applikationen

Kör backend och frontend i separata terminaler

### Terminal 1 - Backend
```bash
cd Backend
node server.js
```

### Terminal 2 - Frontend
```bash
cd frontend
npm run dev
```

## Öppna applikationen
Gå till http://localhost:5173 i din webbläsare
